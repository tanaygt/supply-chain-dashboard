## 📊 Dashboard Features

| Module | Description |
|---|---|
| **KPI Summary** | Fulfillment rate, avg delivery days, low-stock alerts, supplier rating |
| **Inventory Analysis** | Stock levels by category, items below reorder point, turnover table |
| **Order & Logistics** | Monthly order trends, revenue tracking, order status, regional delivery performance |
| **Supplier Scorecard** | Ratings, lead times, defect rates, contract values |
| **Demand Forecasting** | 3-month Moving Average forecast with confidence interval per product |

## ⚡ Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/tanaygt/supply-chain-dashboard.git
cd supply-chain-dashboard
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the dashboard
```bash
streamlit run app.py
```

The dashboard will open at `http://localhost:8501` in your browser.

---

## 📈 Key Metrics Tracked

- **Order Fulfillment Rate** — % of orders delivered vs total orders
- **On-Time Delivery %** — orders delivered within SLA (7 days)
- **Inventory Turnover Rate** — how fast stock is sold/used
- **Supplier Defect Rate** — quality control KPI per vendor
- **Demand Forecast Accuracy** — 3-month forward prediction with confidence band

---

## 💡 Business Use Cases

1. **Procurement teams** can identify which suppliers have high defect rates and long lead times
2. **Warehouse managers** can see which SKUs are below reorder point before stockouts occur
3. **Logistics teams** can track regional delivery performance vs SLA targets
4. **Demand planners** can use the forecast module to prepare purchase orders 3 months in advance

---

## 🗂️ Sample Data

All data is synthetically generated by `data_generator.py` with realistic distributions:
- 500 orders across 5 regions and 12 months
- 25 products across 5 categories
- 15 suppliers with lead times, defect rates, and ratings
- Demand forecast using 3-month moving average with seasonal adjustment
